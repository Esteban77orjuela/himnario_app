# ADR-002: Navegación con React Navigation

## Contexto
App móvil con 4 tabs principales y pantallas hijas (detalle de canción, detalle de setlist).

## Decisión
Usar React Navigation con:
- **Stack Navigator** raíz (`AppNavigator.tsx`) que contiene los tabs y las pantallas modales
- **Bottom Tab Navigator** (`BottomTabNavigator.tsx`) para las 4 secciones principales

### Estructura
```
Stack (AppNavigator)
├── BottomTabs
│   ├── HomeScreen (Himnario)
│   ├── ImportScreen
│   ├── SetlistsScreen
│   └── SettingsScreen
├── HymnDetailScreen (modal stack)
└── SetlistDetailScreen (modal stack)
```

### Tipo de navegación
- `@react-navigation/native-stack` (nativo, no JS) para mejor performance
- Parámetros tipados con `route.params` en cada screen

## Consecuencias
- Las pantallas de detalle están fuera de los tabs, permitiendo que la tab bar desaparezca
- No se usó `@react-navigation/drawer` porque la app solo tiene 4 secciones
- No se usa navegación profunda (deep links) — no es necesaria para una app 100% offline
